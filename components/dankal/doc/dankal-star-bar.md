# Dankal-Star-Bar(星级选择)
## 示例

```html 

 <dankal-star-bar
      v-model="star"
      :read-only="true"
      :quantity="6"/>

```
## Props配置

|  参数 |  说明 |  类型	 |  是否必传	 |  默认值 |
|---|---|---|---|---|
|  value | 选中星星数量  |  Number |  是 |  0 |
|  quantity | 总数量  |  Number |  否 |  5 |
| readOnly|只读 |Bollean | 否| false
